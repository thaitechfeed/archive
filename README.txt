# วิธีการใช้งานคลังทวิตเตอร์ของคุณ

วิธีใช้คลังข้อมูลทวิตเตอร์ของคุณที่ง่ายที่สุดคือการเรียกใช้คลังข้อมูลทวิตเตอร์ผ่านหน้าเว็บที่เตรียมไว้ให้ในไฟล์นี้ เพียงดับเบิ้ลคลิกที่ไฟล์ `index.html` จากโฟลเดอร์บนสุด คุณก็จะสามารถสํารวจประวัติการทวีตของคุณทั้งหมดผ่านโปรแกรมเบราเซอร์ของคุณเลย

---

ในโฟลเดอร์ `data` คลังทวิตเตอร์ของคุณจะปรากฎในสองรูปแบบคือ JSON และ CSV แยกตามเดือนและปี

* CSV เป็นรูปแบบมาตรฐานที่สามารถนําเข้าไปยังเครื่องมือหลายๆ ตัวได้ เช่น โปรแกรมสเปรดชีท หรือถูกเรียกใช้ง่ายๆ ด้วยภาษาโปรแกรมมิ่งทั่วๆ ไป

## JSON สําหรับนักพัฒนา

* ข้อมูลรูปแบบ JSON ประกอบด้วยข้อมูลทวีตรูปแบบเต็มของคุณตาม API รุ่นที่ 1.1 ของทวิตเตอร์คืนค่าไปให้ ดูเพิ่มได้ที่ https://dev.twitter.com/docs/api/1.1
* ข้อมูลรูปแบบ JSON ยังถูกใช้สําหรับการนํามาแสดงผลผ่านหน้าเว็บในไฟล์ index.html อีกด้วย
* เพื่อนําข้อมูล JSON ไปใช้งานในโปรแกรมทั่วไป ให้ลบบรรทัดแรกและบรรทัดสุดท้ายของแต่ละไฟล์ออก

เพื่อให้การตอบรับ ถามคําถาม หรือแบ่งปันความคิดใหม่ๆ กับนักพัฒนาทวิตเตอร์รายอื่นๆ เข้าร่วมฟอรั่มสนทนาได้ที่ https://dev.twitter.com
